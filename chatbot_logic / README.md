# AI Chatbot Logic Engine

## Description
This project implements a chatbot using rule matching and unification.

## Algorithm
1. Take user input
2. Match input with predefined rules
3. Apply unification (variable matching)
4. Generate response

## Example
Input: parent(John,Mary)
Output: child(Mary,John)

## Concepts Used
- Unification
- First Order Logic
- Rule-based AI
## Algorithm Used
Unification Algorithm

## Description
The chatbot uses rule matching and unification to match user queries
with predefined logical rules.

## GUI
Implemented using Tkinter

## Sample Input
hello(John)

## Output
User Input: hello(John)
Matched Rule: hello(X)
Response: Hello John! How can I help you?
